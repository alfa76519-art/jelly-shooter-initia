# 🍬 Jelly Shooter - Initia L2 Edition

### 🎮 How to Play
1. Launch the game via [Vercel](https://jelly-raffle-jangkung.vercel.app).
2. Shoot candies and defeat the Donut Boss to earn high scores.
3. Connect your Keplr wallet and reach 2100+ points to unlock the "Claim" button.

### ⚙️ Technical Setup
- The game communicates with a local Ubuntu relayer via a secure Ngrok tunnel.
- Transactions are broadcasted using `BROADCAST_MODE_SYNC` to the `initiation-2` chain.
